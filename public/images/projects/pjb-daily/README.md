# PJB Daily Image Assets

This folder is reserved for approved PJB Daily promotional artwork and screenshots.

Expected filenames:

- `hero.png`
- `calendar-home.jpg`
- `daily-agenda.jpg`
- `task-editor.jpg`
- `creation-modal.jpg`
- `note-editor.jpg`
- `busy-calendar.jpg`
- `dark-mode.jpg`

Do not add placeholder or fake screenshots. Only add final approved product images.

Images that include usernames, email addresses, account details, or unapproved
store availability claims should remain unpublished until anonymized versions are
available.
